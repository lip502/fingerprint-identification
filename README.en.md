# MATLAB指纹识别（GUI，指纹入库，论文）

#### Description
该课题为基于MATLAB的指纹识别系统，带有GUI可视化界面，流程为：逐一录入指纹，对每个指纹进行特征值提取，输入待匹配指纹，得出是否在库里面，输出结果。带有论文和GUI界面。
